Title: EMail Checker Updated (from mutiple POP3 accounts)
Description: checks your new mails from mutiple mail servers. it switches tray icons on different states & displays the number of new messages (as msn messenger display messages) and plays a WAV file.
new updates:
1- use registry to save configurations. no need to type user name or server name etc every time you start application. it just starts automatically(except first time).
2- define delay/check new mails at any time.
3- Pause/play mail checking application.
4- check new mails from a sigle mail server out of saved mail servers.
Note: in the next version user i'm trying to run default email application for poping and sending mails. any information regarding sending and poping email from the default email application will be apriciated. please vote me if you like it.
